### الطريقة 1: نسخ والصق
```
1. افتح ملف SQL من المجلد
2. انسخ المحتوى كاملاً
3. الصق في Supabase SQL Editor
4. اضغط "Run"
```

### الطريقة 2: استخدام CLI (متقدم)
```bash
supabase db push < SQLs/supabase-setup.sql

```
