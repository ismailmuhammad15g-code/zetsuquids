# ZetsuGuide - Developer Knowledge Base

A modern, minimalist full-stack application for developers to save and organize their important information, code snippets, AI prompts, and development guides.

---

## Features

### 🎯 Prompts Management
- Save AI prompts you frequently use
- Quick copy to clipboard
- Smart search with AI fallback
- Tag-based organization
- Highlight search matches in yellow

### 📚 Guides System
- Import HTML guides into your knowledge base
- Auto-indexing with **Index New!** feature
- Full-text search across all guides
- Keyword extraction and categorization
- Beautiful guide viewer

### 🎨 Design
- Clean black & white aesthetic
- Minimalist, professional design
- Subtle animations
- Fully responsive
- Enterprise-grade UI

---

## Tech Stack

- **Frontend**: React 18 + Vite
- **Backend**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router v6
